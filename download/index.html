The most recent release from the current series:

http://openvswitch.org/releases/openvswitch-2.6.1.tar.gz

The most recent release from the LTS series:

http://openvswitch.org/releases/openvswitch-2.5.1.tar.gz

You can also view the complete list of Open vSwitch releases.

You can also generate your own source zip by clicking "Download ZIP" on a commit in our github project page.  For a snapshot tarball of the most recent commit on the main development branch just use the following link:

https://github.com/openvswitch/ovs/tarball/master

For a table that shows the Linux kernel versions with which the kernel module included in various Open vSwitch versions is compatible, please see the FAQ answer to "What Linux kernel versions does each Open vSwitch release work with?".

Once you've obtained the release of your choice, see the documentation page for more information on how to build, install, and use the software.  If you would like to contribute to openvswitch, full instructions for using git are available on the development page.
